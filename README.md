## SITE: https://caiopolak.github.io/doutorcelularinformatica_site/

## 🚀 O Projeto

O foco principal do projeto foi a criação de uma página de alta velocidade, focada em SEO local e captação direta de clientes via WhatsApp. Diferente de aplicações pesadas com dezenas de bibliotecas, este projeto abraça o **desenvolvimento Vanilla**, atestando o domínio dos fundamentos web (HTML semântico, CSS avançado e JavaScript para interações chave).

### Principais Objetivos Alcançados:
- **Design Premium**: O layout foi criado no modo "*Light Theme*" intersecando tons pastéis (Puro Branco & Cinza Suave) com o poderoso **Amarelo** da marca (`#FFB300`), criando uma hierarquia visual de contraste excepcional.
- **Glassmorphism e Sombras**: Utilização de fundos translúcidos (`backdrop-filter`) na navbar e sombras suaves em vez de contornos rígidos, proporcionando uma sensação moderna e limpa.
- **Responsividade Mobile-First**: O site é totalmente fluído. Adapta suas colunas (CSS Grid & Flexbox) e navegações em dispositivos móveis perfeitamente, garantindo que o usuário de celular tenha a melhor experiência possível.
- **Integração Ágil com WhatsApp**: Botões dinâmicos com chamadas para ação (CTA) que já direcionam interessados com uma mensagem pré-preenchida para o dono do negócio.
- **Animações (Micro-interações)**: Animações de Scroll-Reveal (surgimento sob rolagem) e micro-interações ("Glow" ou pulsação) nos botões, implementadas inteiramente puras em CSS e ativadas via JavaScript DOM-Observer.

## 🛠 Tecnologias Utilizadas

- **HTML5**: Semântica estrita para SEO (tags como `<header>`, `<section>`, `<nav>`, `<footer>`).
- **CSS3 (Vanilla)**: 
  - Variáveis Nativas (`:root`) para consistência do *Design System*.
  - CSS Grid e Flexbox para layouts complexos de ponta a ponta sem frameworks.
  - Animações exclusivas (@keyframes).
- **JavaScript (Vanilla)**:
  - Interação contínua de menu hambúrguer para mobile.
  - IntersectionObserver simulado para revelar seções conforme a rolagem.
  - Funcionalidade de manipulação de botões com URLs dinâmicas para WhatsApp.
- **Phosphor Icons**: Biblioteca de ícones leve e estilosa.
- **Google Fonts**: Inter & Outfit.
